① 建立 Supabase 帳戶
1. 到 supabase.com → 點「Start your project」
2. 用電郵註冊（免費，無需信用卡）
3. 建立新 Project，名稱填 baby-nutrition，地區選 Singapore
4. 等待約1分鐘 Project 完成建立
② 建立資料表（複製以下代碼）
進入 Project → 左欄點「SQL Editor」→ 貼上以下內容 → 點「Run」
create table baby_log (
  id text primary key default 'main',
  log jsonb default '{}',
  notes jsonb default '{}',
  updated_at timestamptz default now()
);
insert into baby_log (id) values ('main') on conflict do nothing;
alter table baby_log enable row level security;
create policy "allow_all" on baby_log for all using (true);
③ 填入連接資料
左欄點「Project Settings」→「API」→ 複製以下兩項：
位置：Settings → API → Project URL
位置：Settings → API → Project API keys → anon public
